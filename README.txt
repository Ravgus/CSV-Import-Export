1. Необходимо создать базу данных, перед запуском проекта, это можно сделать 2-мя способами:
а) запустить файл create_db.php, в корне проекта, который автоматически сформирует бд, и таблицу в ней (заранее необходимо задать данные пользователя бд, и т.д. в конфигурационном файле configs/database.ini).
б) в случае каких либо проблем с 1 вариантом, в корне проекта лежит дамп бд testwork.sql.

Структура проекта:
/configs - содержит конфигурационные файлы.
/controllers - содержит контроллеры приложения.
/core - содержит необходимые, для работы системные классы (роуты, сессии, класы для рендеринга tmp (view) файлов, и т.д.).
/js - содержит пользовательские js файлы
/models - содержит модели приложения.
/services - содержит дополнительные классы (валидация, чтение/запись csv файлов).
/tmpl - содержит шаблоны отображения.

Запуск приложения начинается с файла index.php

P.S. Проект создавался на базе PHP 7.1/MySQL 5.7
